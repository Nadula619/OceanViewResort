📖 System User Guide

Follow these guidelines to manage the OceanView Resort Reservation System efficiently.

🏨 Room Management

  • Monitor current occupancy in the Overview tab.
  • Use Room Availability to check which rooms are ready for new guests.
  • Update room status to MAINTENANCE if a room needs cleaning or repairs.

📝 Guest Bookings

  • Click + New Booking to start the registration process.
  • Search for Existing Guests by name or email to save time.
  • Register new guests by filling out their ID/Passport and contact details.
  • Total price is calculated automatically based on the room rate and stay duration.

💳 Payments & Billing

  • A payment record is created automatically as soon as a booking is confirmed.
  • Go to Payments History to find any transaction.
  • Search by Ref ID (Transaction ID) or Guest Name.
  • Click Generate Bill to view and print the official invoice for the guest.

🆘 System Support

If you encounter technical issues or need access to restricted features, please contact the System Administrator.
